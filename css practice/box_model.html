Box-sizing property

Understand the box model
with margin and padding and border

This box-sizing property tells the browser what the sizing property should include.We can set this property to content-box(default) or border-box.The box-sizing property is used to include the padding and border in an element's total width and height.

content-box:- The width and height properties include only the content.border,padding and margin are not included.

border-box:- The width and height properties includes content,padding and border, but not the margin.
