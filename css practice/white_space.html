Text properties

White-Space

it specifies how white space is handled inside a html element.we can set this property to normal,pre,nowrap,pre-wrap,pre-line.normal:- sequence of white space is collapsed into a single whitespace.text will wrap when necessary.pre:- whitespace is preserved by the browser.text will only wrap on line breaks.it is just like pre tag in html. nowrap:- sequence of white space is collapsed into a single whitespace.Text will never wrap to the next line.The text continues on the same line until the br tag is encountered. pre-line:- sequence of white space is collapsed into a single whitespace.text will only wrap when necessary and on line breaks. pre-wrap:- whitespace is preserved by the browser.text will only wrap when necessary and on line breaks.
