Attribute selector

we can set style using attribute or its value.

set style using attribute- syntax:- selector [attribute] {css};

set style using attribute and its value.syntax:- selector [attribute="value"] {css};

set style using an attributor value containing a specified word. syntax:- [attribute ~="value"]{css}

set style using with the specified attribute starting with the specified value. syntax:- [attribute |="value"] {css};

set style whose attribute value begins with a specified value.It is not necessary that value should be a whole word.syntax:- [attribute ^="value"] {css};

set style whose attribute value ends with a specified value.It is not necessary that value should be a whole word.syntax:- [attribute $="value"] {css};
