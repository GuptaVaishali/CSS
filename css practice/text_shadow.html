Text properties

Text-shadow

it specifies shadow effect added to the text. in a block.we can add more than one shadow to the text,seperated by comma.

syntax:- text-shadow:h-shadow v-shadow blur-radius color

h-shadow:-the position of the horizontal shadow.negative values are allowed. v-shadow:-the position of the vertical shadow.negative values are allowed. blur-radius:-the blur radius.the default value is 0. color:- the color of the shadow.
