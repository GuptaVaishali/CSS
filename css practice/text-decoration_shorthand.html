text-decoration-line values can be underline,overline,line-through,none.we can combine more than one value like underline and overline to one text.

text-decoration-style:-this property is used to specify how the line will display.this property can have values solid,double,dotted,dashed,wavy.

text-decoration-color:- this property is used to specify yhe color of the line.

text-decoration_shorthand:- this is shorthand of all three text-decoration properties. syntax:- selector { text-decoration:text-decoration-line text-decoration-style text-decoration-color}.
