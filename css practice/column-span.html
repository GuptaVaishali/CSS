Column-span property

This property is used to define columns.

column-count:- This property is used to specify the number of columns an element should be divided into. we can set this property to number or auto(default). column-width:- this property to used to specify width for the columns.We can set this property to length or auto(default). columns:- This is a shorthand which is used to set column-count and column-width. syntax:-selector{columns:column-width column-count;}
