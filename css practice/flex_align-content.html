This property is used to modify the behaviour of the flex-wrap property.It is similar to align-items, but instead of aligning flex items, it aligns flex lines.There must be multiple lines of items for this property to have any effect.We can set this property to stretch(default),flex-start,flex-end,center, space-between,space-around.

stretch:- lines stretch to take up the remaining space.

flex-start:- This is used to position the item at the begining of the container.

flex-end:- This is used to position the items at the end of the container.

center:- this is used to position the items at the center of the container.

space-between:- this is used to position the item with space between the lines.

space-around:- This is used to position the item with space before, between, and after the lines.
