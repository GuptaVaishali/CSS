Border properties

Border_shorthand

This property sets border style of an element.We can set this property to none,hidden,dotted, dashed,double,solid,groove,ridge,inset,outset.

it sets width of element border.we can set this property to thin,medium,thick,length in the form of px.

it defines the color of border.we can set this property to tranparent or any color value.

it is a shorthand of all border properties.the order should be border-width,border-style,border-color. Border-style is mandatory.
