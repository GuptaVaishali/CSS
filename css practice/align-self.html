This property is used to specify the alignment for the selected item inside the flexible container.The align-self property overrides the flexible container's align-items property.We can set this property to auto(default), flex-start,stretch,flex-end,center,baseline.

auto:- The element inherits its parent container's align-items property ,or stretch if it has no parent container.

flex-start:- This is used to position the item at the begining of the container.

flex-end:- This is used to position the items at the end of the container.

center:- this is used to position the items at the center of the container.

stretch:- this is used to stretch the item to fit the container.

baseline:- This is used to position the item at the baseline of the container.
