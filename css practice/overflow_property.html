overflow property

Start...This property is used to define what happens if content overflows an element's box.It means it will define whether to clip content or to add scroll bars when an element's content is too big to fit in a specified area.This property only works for block elements with a specified height.We can set this property to visible(default),hidden,auto,scroll.
